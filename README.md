## ğŸ¨ Customization
![Settings Example](https://github.com/hakhangthu7558-sys/Coca-Cola-Christmas/blob/main/img-pic/workbench.png?raw=true)
You can customize the theme colors in your VS Code settings:

```json
{
    "workbench.colorCustomizations": {
        "[Coca-Cola Christmas]": {
            "editor.background": "#0d0d0d",
            "editor.foreground": "#ffffff"
        }
    },
    "editor.tokenColorCustomizations": {
        "[Coca-Cola Christmas]": {
            "comments": "#d0d0d0",
            "strings": "#006400"
        }
    }
}
```
